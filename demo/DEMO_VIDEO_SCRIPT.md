# BCS Website Demo Video Script & Recording Guide

Complete script and instructions for creating a professional demo video showcasing all features of the Bellevue Collision Services website.

**Target Length**: 5-7 minutes
**Format**: MP4, 1080p (1920x1080)
**Audience**: Potential customers and business owner

---

## Recording Setup

### Software Needed
- **Screen Recording**: OBS Studio (free) or Loom
  - Download OBS: https://obsproject.com/
  - Download Loom: https://www.loom.com/
- **Video Editing**: DaVinci Resolve (free) or iMovie
- **Audio**: Built-in microphone (or external for better quality)

### Recording Settings
```
Resolution: 1920x1080 (1080p)
Frame Rate: 30 FPS
Audio: 48kHz, stereo
Format: MP4 (H.264)
Bitrate: 5000 kbps (video), 192 kbps (audio)
```

### Browser Setup
- Use Chrome or Firefox in Incognito/Private mode
- Clear all cookies and cache
- Zoom level: 100%
- Hide bookmarks bar
- Full screen browser (F11)
- Close unnecessary tabs

## Recording Instructions

### Pre-Recording Checklist
- [ ] Clear browser cache and cookies
- [ ] Prepare test data (sample customer, appointments, repairs)
- [ ] Set up admin account with demo data
- [ ] Have damage photos ready for upload
- [ ] Test microphone audio levels
- [ ] Close unnecessary applications
- [ ] Set "Do Not Disturb" mode
- [ ] Ensure good lighting (if showing face)
- [ ] Have script printed or on second screen

### Recording Steps

#### Using OBS Studio:

1. **Set up OBS**:
   - Add Display Capture source
   - Set resolution to 1920x1080
   - Add Audio Input Capture (microphone)
   - Set recording format to MP4

2. **Recording**:
   - Start recording (hotkey or button)
   - Follow storyboard scene by scene
   - Speak clearly and at moderate pace
   - Pause between major sections (can edit later)
   - Stop recording when complete

3. **Save Recording**:
   - File will be in OBS recordings folder
   - Default: `~/Videos/`

#### Using Loom:

1. **Set up Loom**:
   - Install Loom desktop app or Chrome extension
   - Sign up for free account
   - Select "Screen + Camera" or "Screen Only"

2. **Recording**:
   - Click "Start Recording"
   - Select which screen to record
   - Begin narration and demonstration
   - Click "Stop Recording" when done

3. **Download**:
   - Loom automatically uploads
   - Click "Download" button
   - Select MP4 format, 1080p

## Post-Production Editing

### Basic Edits (Optional):

1. **Trim**:
   - Cut out any mistakes or long pauses
   - Remove dead air at start/end

2. **Add Title Screen**:
   - First 3 seconds: Company logo + website URL
   - Fade in from black

3. **Add End Screen**:
   - Last 5 seconds: Contact information
   - Fade to black

4. **Add Background Music** (optional):
   - Use royalty-free music at low volume
   - Sources: YouTube Audio Library, Epidemic Sound

5. **Color Correction**:
   - Adjust brightness if needed
   - Ensure website colors look accurate

### Using DaVinci Resolve (Free):

1. Import video clip
2. Drag to timeline
3. Use Cut tool to trim sections
4. Add titles from Effects Library
5. Export as MP4:
   - Format: MP4
   - Codec: H.264
   - Resolution: 1920x1080
   - Frame rate: 30
   - Quality: High (10,000 kbps)

## Tips for Best Results

### Audio:
- Speak clearly and at moderate pace
- Smile while speaking (it shows in your voice)
- Pause briefly between sentences
- Re-record sections if you make mistakes
- Use external microphone if possible

### Video:
- Move mouse slowly and deliberately
- Don't rush through screens
- Pause on important information
- Use smooth scrolling (not jerky)
- Zoom in on small text if needed

### Content:
- Show don't tell - let actions speak
- Keep it concise (under 7 minutes ideal)
- Focus on benefits, not just features
- Show real-world scenarios
- End with clear call-to-action
